ng-app-skeleton
===============

Prerequisites
------

[Node]
[Grunt]
[Yeoman]

How to
------

Create a new folder, then go to the new directory. Replace "ng-app" by your "app-name"
```sh
mkdir ng-app && cd $_

git clone https://github.com/EdisonDelgado/ng-app-skeleton.git

```

Install Dependencies

```sh
npm install
bower install
```

Start server
```sh
grunt serve
```
