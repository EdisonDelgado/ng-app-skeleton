ng-app-skeleton
===============

Prerequisites
------

[Node]
[Grunt]
[Yeoman]

How to
------

Clone this project. Rename "ng-app-skeleton" to app name "my-app"
```sh

git clone https://github.com/EdisonDelgado/ng-app-skeleton.git

mv ng-app-skeleton my-app

cd my-app

```

Install Dependencies

```sh
npm install
bower install
```

Start server
```sh
grunt serve
```
